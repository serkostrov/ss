# Как узнать ID группы в Telegram и Max

Привязка чата к рабочей группе: **Админ → Группы → [группа] → Чаты**.

## Telegram

1. Создайте группу (или канал) и добавьте в неё бота АПСС как участника/администратора.
2. Узнайте `chat_id`:
   - добавьте в группу бота `@userinfobot` или `@getidsbot` — он пришлёт id (часто вида `-100…`);
   - либо перешлите любое сообщение из группы боту `@RawDataBot` и скопируйте `chat.id`.
3. Вставьте значение в поле **ID чата** при привязке.

## Max

1. Создайте чат/группу в Max и добавьте бота АПСС.
2. Откройте свойства беседы в приложении или кабинете Max для бизнеса и скопируйте идентификатор чата (`chat_id`).
3. Вставьте значение в поле **ID чата**.

После сохранения статус подключения можно выставить вручную (`pending` / `connected` / `error`) до готовности worker.
